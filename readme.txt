# Single Page CV

This is a  Single Page CV  project built with only **HTML** and **CSS**. It provides a clean, responsive layout to showcase your resume effectively on a single page.



## 📄 Features
- Fully responsive design for desktops, tablets, and mobile devices.
- Clean and professional layout.
- Easy to customize with your personal information.

## 🛠️ Technologies Used
- **HTML**: To structure the content.
- **CSS**: To style and design the page.
